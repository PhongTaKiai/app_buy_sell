# AppBuySell

## 概要
AppBuySellは個人開発プロダクトをアプリ内で簡単に売買することを目的としたFlutterアプリケーションです。

## 特徴
- アプリ内で簡単に売買
- 購入と同時に権限移譲
- 掲載も簡単
